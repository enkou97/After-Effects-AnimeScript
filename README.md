# After Effects 素材自動インポートスクリプト

## 概要
撮影監督の要件をもとに、素材フォルダから自動的に使用可能なファイルを選別し、
After Effectsに自動で読み込むスクリプトです。

## 使用技術
- JavaScript (Adobe ExtendScript)
- After Effects Scripting API
- Regexによるファイルフィルタリング

## 特徴
- 素材の自動選別（ファイル拡張子チェック）
- 合成作成、キャッシュ削除の自動化
- AEプロジェクトへの自動インポート

## 開発環境
- VSCode
- Adobe ExtendScript Toolkit
- OS: Windows10 / macOS Ventura
